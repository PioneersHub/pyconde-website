Universal Rules

  • White text works on all 6 primary colors and dark secondaries (Coral, Dark Red, Magenta).
  • Blue #3778be is the most versatile dark text color.
  • Light pastels (#e199c3, #a2d9ef, #d5f199) — use as backgrounds only, rarely as text.
  • Similar-brightness colors always fail (e.g., Yellow on Orange, Lime on Cyan).
  • When in doubt: check WCAG AA contrast ratio (minimum 4.5:1 for body text, 3:1 for large text).

3. Typography

Single font family: Helvetica Neue. All text is regular (upright) — not italic. Two weights only: Regular and Bold. No exceptions. Fallback: IBM Plex Sans.

5. Illustration System — Bars & Pythons

Color Bars

7 horizontal bars, each X in height, stacked directly. Minimum margin from edge: at least X. The color order is fixed and must not be altered.

Body-Part Vocabulary (5 Shapes)

Each shape fits within one X × X module. These are the building blocks for all python illustrations.

Python Construction Rules

  • Pythons may freely wriggle in any direction using smooth curves.
  • Maximum straight section: 4 body parts (4 × X). Never longer.
  • No sharp edges — all turns must use quarter-circle or semicircle bends.
  • Body color matches the bar it passes through, creating multicolored banding.
  • Each python has a semicircular head dome with a contrasting eye dot.
  • Each python has a leaf-shaped tail tapering to a point.

8. Pixel Art Photo Treatment

Photographs are processed into pixel-art style using only brand palette colors. Each pixel maps to the nearest brand color, creating a retro-digital aesthetic unique to the conference.

Usage

  • Speaker portraits on website and social media
  • Conference badges and name tags
  • Promotional materials and merchandise

10. Lightning Bolt Motif

A geometric lightning bolt shape used as a decorative/accent element. Available in two renderings:

⚡
Flat Version Solid flat colors: Magenta upper portion transitioning to Yellow/Orange tip. Background: horizontal bands of Blue, Cyan, White, Grey. Semi-transparent overlapping rectangles.
⚡
Gradient Version Smooth gradient: Purple to Magenta to Orange to Yellow. Bottom band changes to Lime Green. More dynamic, polished feel.

Usage Rules

  • Uses brand colors exclusively — no off-palette colors
  • Semi-transparent overlapping areas are part of the design language
  • Suitable for backgrounds, section dividers, and accent graphics

See Appendix pages 17–18.

12. Quick Reference — Do's & Don'ts

Do

  • Always set text in Helvetica Neue Regular (upright), Bold or Regular weight
  • Maintain X-unit clear space around the logo on all sides
  • Keep the monogram and wordmark together as one unit
  • Check both primary and secondary color combination tables before pairing
  • Verify WCAG AA contrast ratios when in doubt
  • Build pythons with smooth curves using the 5 modular shapes
  • Limit straight python sections to 4 body parts maximum
  • Use pixel-art treatment for conference photography
  • Vary python head sizes and orientations in group compositions
  • Use the lightning bolt in either flat or gradient style
  • Refer to the original PDF pages for exact artwork

Don't

  • Use any font other than Helvetica Neue
  • All text is regular (upright) — never italic
  • Use any of the 17 forbidden primary color combinations for text
  • Use any of the 14 forbidden secondary color combinations for text
  • Use light pastels as text on medium or light backgrounds
  • Make python straight sections longer than 4 body parts
  • Use sharp angles or corners in python illustrations
  • Colorize the logo — it is always black or white only
  • Separate the monogram from the wordmark
  • Alter the bar color order, proportions, or color combinations
  • Use non-brand colors in the pixel art photo treatment
